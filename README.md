# Teslo api
1. Clonar proyecto

2. instalar dependencias

```
npm i
```

3. Clonar el archivo __.env.template__ y renombrarlo a __.env__

4. Llenar las variables de entorno 

5. levantar la base de datos
```
docker compose up -d
```

6. Ejecutar Seed

```
localhost:3000/api/seed
```

7. levantar el modo desarrollo

```
npm run start:dev
```
